# Gerenciador de Gastos Pessoais

Sistema de controle de gastos feito em Python, com menu interativo no terminal e persistência de dados em arquivo.

## Funcionalidades

- Adicionar gasto (descrição, categoria e valor)
- Ver todos os gastos cadastrados
- Ver o total gasto
- Dados salvos automaticamente em arquivo (não perde ao fechar o programa)

## Como executar

1. Tenha o Python instalado
2. Rode o arquivo pelo terminal:

python gerenciador_gastos.py

## Tecnologias e conceitos utilizados

- Python
- Estruturas de repetição (while, for)
- Estruturas condicionais (if/elif)
- Listas e dicionários
- Tratamento de erros (try/except)
- Manipulação de arquivos (leitura e escrita)
